# Personality Decoder App (ASP.NET Web Forms)

This project is a simple web application built using ASP.NET Web Forms to demonstrate core concepts.

## 📌 Concepts Covered
- Server Controls (TextBox, DropDownList, Button, Label)
- Event Handling
- Code-Behind Logic
- PostBack Mechanism
- Basic Validation

## 🧠 How It Works
1. User enters their name
2. Selects their rashi
3. System processes input using simple logic
4. Personality traits are displayed dynamically

## 🛠 Technologies Used
- ASP.NET Web Forms
- C#
- Visual Studio

## ▶️ How to Run
1. Open project in Visual Studio
2. Build the solution
3. Run the application (IIS Express)

## 🎓 For Students
This project is designed to help beginners understand how ASP.NET Web Forms works through a simple real-world example.
